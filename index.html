A responsive, difficulty-based number guessing game built with React and styled using Bootstrap. The app features a live timer, result tracking stored in localStorage, and a results page to review past attempts. Designed to demonstrate proficiency with React hooks (useState, useEffect), routing, conditional rendering, and user experience flow.
